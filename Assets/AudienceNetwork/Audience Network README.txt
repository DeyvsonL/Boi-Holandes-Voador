Audience Network Unity Plugin README

--------------------------------------

Included Files:
*.unity 				Sample app scenes
*.cs					Sample app code
FBAudienceNetwork.dll	Plugin library
Plugins/*				iOS/Android specific libraries

Getting Started:
Import FBAudienceNetwork.dll into your Unity project, and add everything under Plugins to your Asset folder. Android users may need to edit the Plugins/Android/AndroidManifest.xml. Platforms other than iOS and Android are not supported.
